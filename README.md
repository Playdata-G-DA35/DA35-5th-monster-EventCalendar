# DA35-5th-monster-EventCalendar

## 프로젝트 이름 : Event Calendar

### 프로젝트 주제 : 팀원끼리 일정을 공유하는 웹페이지 제공

### 팀원
- 이도형 : 웹페이지 디자인 관리
- 이서희 : 일정 달력 구현
- 이성재 : HTML 태그 관리
- 하승주 : 계정 관련 기능 구현
